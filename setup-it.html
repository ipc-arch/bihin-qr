AppSheet → Data → Columns で以下の列の Show_If 欄に式を入力。

列名Show_If 式
リース期限[資産区分] = "リース"
リース会社[資産区分] = "リース"
OSバージョンNOT([資産区分] = "消耗品")
シリアル番号NOT([資産区分] = "消耗品")
MACアドレスNOT([資産区分] = "消耗品")
💡 消耗品（トナー等）の入力時にOS欄やシリアル欄が消えるので、入力者が迷わない。

5. OCR型番読み取り

仕組み

AppSheetの Image型 列 + App Formula を使って、撮影した型番シールからテキストを抽出します。

⚠️ AppSheetのOCR機能は限定的です。現状、完全自動のテキスト抽出はAppSheet単体では難しい。以下の2段階アプローチが現実的：

方式A（推奨）: 写真撮影 + 手入力の半自動化

実運用ではこれが一番確実で速いです。

  1. 「型番写真」列（Image型）をタップ → iPadカメラが起動
  2. 備品裏面の型番シールを撮影 → 自動でDrive保存
  3. 撮影した写真を見ながら「型番」「シリアル番号」列に手入力
💡 型番写真が残るので、後から「あれ、型番なんだっけ?」が無い。写真がそのままエビデンスになる。

方式B（発展）: Google Apps Script + Cloud Vision OCR

完全自動化したい場合の将来オプション。実装コストがあるので、まずは方式Aで十分。

6. リース期限アラート

6-1. Format Rules で色分け

UX → Format Rules → + New Format Rule

ルール名条件式色

💡 IT-0003（SHARP複合機、2026-03-31期限）が黄色で表示されたら、フォーマットルールが正常に動作しています。

6-2. リース期限切れ通知（オプション・GAS）

将来、Google Apps Script で毎月自動メール通知を飛ばすことも可能。まずは色分けだけで運用。

7. 個人情報の取り扱い

⚠️ 「貸出先個人」列には職員の氏名が入ります。アプリの閲覧権限設定に注意。

推奨設定

閲覧者見える範囲設定方法
システム課（Admin）全列・全行Role = Admin
管財課全列・全行Role = Admin
他課の職員（User）貸出先個人は非表示列のShow_If: IN(USEREMAIL(), Admins[Email])

他課から見えるのは「貸出先部署」まで。個人名はシステム課・管財課のみ閲覧可。
